Legend Lost

☓

The player begins the game by talking with the knight Lancelot and him asking the player to help find areas so he can set up the different activities in the tournament. This requires the player to find areas given certain information.
Once completing all the constructs, Lancelot invites you to join the tournament. The first activity in the tournament is shooting a catapult at a target. To do this, you must find the distance between you and the target thus using numberlines.

Anonymous Suggestion App for Moser Consulting

☓

Moser Consulting wanted to build an app that would enhance employee communication and allows the employees to be heard.

Built a mockup of how the app would run. Put the demo on Invision and allowed fellow Moser employee's to test it and give feedback before development.
Had weekly meetings with my team lead to discuss progess with the app throughout the summer.
This app was built with Java for android devices. Used Google Firebase for the back end.

Project Charter for Moser Consulting

☓

A form based web page for project managers to organize and list information needed to organize upcoming projects. Was used as a demo product for Project Management Consulting/Training courses ran by Moser.

Built a mockup of how the app would run thorugh invision and then took feedback from the person who ran project manager training courses (Lou Russel).
I built this with Ruby on Rails. Through the summer I would give demos of the app to the employee who ran the project manager training courses and got feedback from her.
I presented this app in front of Moser's tech-leads to demo the app for tech leads and show off what I had done.

Pokemon: Awakening (WIP)

☓

Intro screen to the game that any good Pokemon game is sure to have. Gives the title of the game and thus sets the standard for
Menu that allows users to either load their previously saved game or start a new game. The third button doesn't do any thing now but I figured I would leave room for future flexibility if I need another button in the future. For now if you click it, it gives you cool quotes such as "Saw Inception...Or at least I dreamt I did. -Michael Scott."
Just like any Pokemon game, can't start a new game without a classic rolling text screen intro!
You can move your units around the map and battle enemy units. Each unit has two actions each round and once those actions are taken, the unit turns to a stoney color.
