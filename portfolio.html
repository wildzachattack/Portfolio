Legend Lost

☓

Built with two partners. We decided to build an education game for a community center in Marion, Indiana. The community center got a grant to add computers and were looking for ways to use them for education.

We worked alongside an education professor at Indiana Wesleyan. He helped us with our choices of how to effectively teach math concepts.
This minigame is finding the areas of multiple shapes. I programmed the shapes appearing on and off the screen as you progress, the UI, and the checking of the correct answers.
Created the UI for this minigame and ran tests on the different aspects of making sure the random numbers were random and that the absolute value method was working.
Programmed the timer and the UI for this minigame where you race around the track answering math questions. Our Github repository.

Anonymous Suggestion App for Moser Consulting

☓

Moser Consulting wanted to find a way to enhance employee satisfaction and allow the employees voices to be heard, the other intern and I were tasked with finding the solution.

Built a mockup of how we thought an app for the solution would work. Put the demo on Invision and allowed fellow Moser employee's to test it and give feedback to see if it would be an effective tool.
Had weekly meetings with our manager to discuss progess with the app and more possible features that might imporve the app.
This app was built for Android devices. Was built with Java and Google Firebase for the back end.

Project Charter for Moser Consulting

☓

A form based web page for project managers to organize information for upcoming projects. Used as a demo product for project management training courses ran by Moser.

Built a mockup of how the app would run thorugh invision and then took feedback from Lou Russel (she ran project management training).
I built this with Ruby on Rails. Through the summer I would give demos of the app to Lou Russel who would give me suggestions and feedback as I progressed in the project.
I presented this app in front of Moser's tech-leads to show off what I had done through the summer.

Pokemon: Awakening (WIP)

☓

Tactical turn-based game that revolves around strategic movement across grid-based maps. Uses sprites and characters from Nintendo and Game Freak's franchise of Pokemon.

Just like any Pokemon game, can't start a without a classic rolling text screen intro! Sets up the future RPG aspect this game will contain.
You are allowed two actions per unit. Once actions are taken, unit appears with a stone colored tint covering it. Movement and collisions are the true functional aspects working.
Based off the combat style of the video game franchise Fire Emblem. My Github repository for the scripts.
